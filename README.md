# jarfernandez Homebrew Tap

Custom formula repository for [Homebrew](https://brew.sh).

## Usage

Add this tap to Homebrew:

```sh
brew tap jarfernandez/tap
```

Then install any formula from the tap:

```sh
brew install jarfernandez/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [check-image](Formula/check-image.rb) | Go-based CLI tool for container image validation |

## Update the tap

```sh
brew update
```

## Remove the tap

```sh
brew untap jarfernandez/tap
```
